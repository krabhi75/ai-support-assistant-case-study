# Failure Analysis & RCA

## Common Failure Modes
1. Hallucinated responses
2. Outdated documentation
3. Ambiguous user queries

## Root Causes
- Missing or incomplete documents
- Overly broad prompts
- Low confidence threshold

## Mitigation Actions
- Improve document coverage
- Tighten prompt constraints
- Increase fallback sensitivity

## Learnings
It is safer to not answer than to answer incorrectly.
